# INTERNAL LINKING ARCHITECTURE

## Hub-and-Spoke Model

### Level 1: Homepage
Links to ALL Tier 2 Hub Pages (10-12 links)

### Level 2: Category Hubs
- Link to Homepage
- Link to sibling hubs
- Link to all child pages (15-25 links)

### Level 3: Service/Route Pages
- Link to parent hub
- Link to homepage
- Link to related routes (5-8 links)
- Link to city pages (2 links)
- Link to booking/pricing (1 link)

### Level 4: Supporting Pages
- Link to parent service page
- Link to relevant hubs
- Link to related blog posts (3-5 links)
